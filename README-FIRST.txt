JAVAROCK - START HERE
=====================

1. Extract the entire ZIP to a normal folder.
2. Double-click START-JAVAROCK.bat.
3. Read the requirement report. If anything is missing, choose Yes to prepare it.
4. Add your own Microsoft account in the JavaRock window.
5. Select a Bedrock Realm and start the bridge.
6. In Minecraft Java Edition, connect to localhost:25565.

Do not move START-JAVAROCK.bat away from the other files.
JavaRock needs Node.js 20+ and JDK 17+. It does not need Python or Tkinter.
No account, token, Realm name, or Realm id is included in this package.
Use only accounts and Realms you are authorized to access.

For help, visit:
https://github.com/AvacadoWizard120/JavaRock-MC-Realms-Crossplay/blob/main/docs/GETTING_STARTED.md
